## üìå Notes Section

### Pattern Templates:

**Two Pointer:**

```
left, right = 0, len(arr) - 1
while left < right:
    if condition_met:
        # process
        left += 1
        right -= 1
    elif need_larger_sum:
        left += 1
    else:
        right -= 1
```

**Sliding Window:**

```
left = 0
for right in range(len(s)):
    # expand window
    while condition_violated:
        # shrink window
        left += 1
    # update result
```

**Stack (Monotonic):**

```
stack = []
for i, val in enumerate(arr):
    while stack and arr[stack[-1]] < val:
        # process stack.pop()
    stack.append(i)
```

**Tree DFS:**

```
def dfs(node):
    if not node:
        return base_case
    left = dfs(node.left)
    right = dfs(node.right)
    return combine(left, right, node.val)
```

**Graph BFS:**

```
from collections import deque
queue = deque([start])
visited = {start}
while queue:
    node = queue.popleft()
    for neighbor in graph[node]:
        if neighbor not in visited:
            visited.add(neighbor)
            queue.append(neighbor)
```
